# The future of AI is local — and physical

We build AI hardware: machines that run your models, and objects with a mind. Open source, from the BIOS settings to the CAD files.

## Own your AI

- **[autonomous-computer](https://github.com/autonomous-ai/autonomous-computer)** — Build your own Personal AI Computer. Three sizes, from 2× RTX 5090 to 8× GPU. Every part, every bracket, every BIOS setting, every assembly photo.
- **[autonomous-grid](https://github.com/autonomous-ai/autonomous-grid)** — The orchestration layer for local AI. Pools the computers you already own behind one OpenAI-compatible endpoint and routes each request to the right model.

The full stack: build the machine, then serve open-weight models from it. **Build it once; own it for good.**
